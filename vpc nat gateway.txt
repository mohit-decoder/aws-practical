STEP 1: CREATE VPC
	 -VPC ONLY
	 -NAME: MM-VPC
	 -IPV4 CIDR 10.0.0.0/16

STEP 2: CREATE SUBNET:
	VPC ID: MM-VPC
	SUBNET NAME: PUBLIC SUBNET
	AVALI. ZONE: NO PREFERENCE
	IPV4 CIDR :  10.0.0.0/24

	VPC ID: MM-VPC
	SUBNET NAME: PRIVATE SUBNET
	AVALI. ZONE: NO PREFERENCE
	IPV4 CIDR :  10.0.1.0/24
STEP 3: CREATE INTERNET GATE WAY
		NAME MM IGW {ATTACH TO VPC}
STEP 4: CREATE ROUTE TABLE:
		NAME PUBLIC ROUTE-TABLE
		EDIT ROUTE: 0.0.0.0/0  INTERNET GATEWAY
		EDIT SUBNET ASS0CIATION: PUBLIC SUBNET
STEP 5: CREATE NAT GATEWAY
		NAME MM NATGATEWAY
		SUBNET: PUBLIC SUBNET
		CONNECTIVITY: PUBLIC
		ELASTIC IP: ALLOCATE ELASTIC IP
		
STEP 6: CREATE ROUTE TABLE:
		NAME PRIVATE ROUTE-TABLE
		EDIT ROUTE: 0.0.0.0/0  NAT GATEWAY
		EDIT SUBNET ASS0CIATION: PRIVATE SUBNET

	LAUNCH INSTANCE NO.1
	NAME: PUBLIC INSTANCE
	KEY: 	CREATE A NEW KEY
	NETWORK: VPC MM-VPC
		 SECURITY GRP: CREATE NEW
		 ENABLE HTTP
		 
	CREATE INSTANCE
	LAUNCH INSTANCE NO.2
	NAME: PRIVATE INSTANCE

	NETWORK: VPC MM-VPC

		 		  ENABLE HTTP
